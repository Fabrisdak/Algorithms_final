# Algorithms Final Project

Comparison of **dynamic programming**, **greedy heuristics**, and **supervised ML** on **0/1 Knapsack** and **Weighted Interval Scheduling**.

| Folder        | Owner    | Contents                                        |
| ------------- | -------- | ----------------------------------------------- |
| `knapsack/`   | Jiacheng | Knapsack solvers and experiments                |
| `scheduling/` | Fabricio | Interval scheduling solvers, tests, experiments |

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

## Knapsack (Python)

From the **repository root**:

```bash
python knapsack/experiments/run_report_round.py
python knapsack/charts/generate_report_charts.py
```

- Experiment log: `knapsack/experiments/results/report_round_results.csv`
- Figures for the paper: `knapsack/charts/output/*.png` (generated by `generate_report_charts.py`)
- Trained ML models: `knapsack/ml_pipeline/models/*.joblib`

## Scheduling (Python)

From the **repository root**:

```bash
python -m pytest scheduling/tests -q
python scheduling/experiments/run_experiments.py --round both
python scheduling/experiments/generate_charts.py
```

- Experiment log: `scheduling/results/results.csv`
- Figures for the paper: `scheduling/charts/*.png` (generated by `generate_charts.py`)
